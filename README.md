# Face-Clustering-KNN-Lab5
using knn i clustered the faces of faculty members (using a picture) and then predicted the cluster into which dr.shashi tharoor's face would be categorised into

Aim:
Detect faces in a picture and cluster them based on skin tone using HSV features.

Method:
1.Used Haar cascade to do face detection
2.extracted Hue and Saturation
3.KMeans Clustering
4.Predict the cluster for Dr.Shashi tharoor's face
